# 超级玛丽 · 网页版

纯原生 HTML5 Canvas + WebAudio 实现的超级玛丽风格平台跳跃游戏,无任何外部依赖与资源文件。

## 运行

```bash
python3 -m http.server 8000
```

然后浏览器打开 <http://localhost:8000>(也可以直接双击 `index.html` 打开)。

## 操作

| 按键 | 功能 |
| --- | --- |
| ← → 或 A D | 左右移动 |
| 空格 / W / ↑ | 跳跃(长按跳得更高) |
| Shift | 加速跑 |

移动端自动显示触控按钮。

## 玩法

- 顶 `?` 方块获得金币,部分方块藏有蘑菇,吃蘑菇变大
- 变大后可以顶碎砖块;受到伤害则变小(有短暂无敌时间)
- 踩扁板栗仔得分,被碰到(小状态)损失一条命
- 金币每集满 100 个奖励 1 条命
- 跨越三个深坑,爬上终点阶梯摸到旗杆即通关,剩余时间折算奖励分
- 时间耗尽或掉坑会损失生命,3 条命用完游戏结束

## 文件结构

- `index.html` — 页面骨架与 HUD
- `style.css` — 界面样式与触控按钮
- `game.js` — 游戏全部逻辑(物理、关卡、像素画精灵、音效)
