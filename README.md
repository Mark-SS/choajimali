# 超级玛丽 · 网页版

纯原生 HTML5 Canvas + WebAudio 实现的超级玛丽风格平台跳跃游戏,无任何外部依赖与资源文件。

## 运行

```bash
python3 -m http.server 8000
```

然后浏览器打开 <http://localhost:8000>(也可以直接双击 `index.html` 打开)。

## 操作

| 按键 | 功能 |
| --- | --- |
| ← → 或 A D | 左右移动 |
| 空格 / W / ↑ | 跳跃(长按跳得更高) |
| Shift | 加速跑 |

移动端自动显示触控按钮。

## 玩法

- 顶 `?` 方块获得金币,部分方块藏有蘑菇,吃蘑菇变大
- 变大后可以顶碎砖块;受到伤害则变小(有短暂无敌时间)
- 踩扁板栗仔得分(蓝色变种速度更快),被碰到(小状态)损失一条命
- 金币每集满 100 个奖励 1 条命
- 爬上终点阶梯摸到旗杆即过关,剩余时间折算奖励分
- 时间耗尽或掉坑会损失生命,3 条命用完游戏结束

## 关卡(共 6 关)

| 关卡 | 主题 | 特色 |
| --- | --- | --- |
| 1 | 草原 | 经典开局,熟悉操作 |
| 2 | 地下洞窟 | 蓝砖洞穴,顶部封闭,悬浮平台 |
| 3 | 星空夜行 | 七段地面、六个深坑,跳跃考验 |
| 4 | 冰雪滑道 | 地面打滑,刹车距离变长 |
| 5 | 黄昏沙漠 | 金字塔阶梯阵,快速敌人增多 |
| 6 | 熔岩城堡 | 最终关:岩浆深坑、低顶通道、敌人最密 |

## 文件结构

- `index.html` — 页面骨架与 HUD
- `style.css` — 界面样式与触控按钮
- `game.js` — 游戏全部逻辑(物理、主题、6 个关卡定义、像素画精灵、音效)
- `check_levels.mjs` — 关卡数据自检脚本(`node check_levels.mjs`)
